Ключевые результаты недели

Исполнение

За отчетный период выполнен комплекс аналитических, методических и организационных работ по направлению централизации лабораторной службы.

  • Презентация для еженедельных совещаний с заместителем Председателя Правительства Московской области — министром здравоохранения Московской области М.В. Забелиным.
  • Целевые показатели проекта на дашборд.
  • Чек-лист для аудита лабораторной службы.
  • Файл приоритизации по лаборатории.
  • Анализ кадровой структуры ЦКДЛ: выгрузка штатного расписания 1С, анализ выгрузки, сводная презентация по кадровой структуре в разрезе ЦКДЛ.
  • Отработка 12 жалоб по МСЭД.
  • ТПР дашборд.
  • Выгрузка госпитальных комплексов ЕМИАС: аудит структуры БД ЕМИАС, формирование ТЗ для выгрузки.
  • ТЗ на выгрузку внешних назначений за 2026 год.
  • Памятка для врачей по назначению исследований в ЦКДЛ.
  • Распределение лабораторно-диагностических исследований по уровням выполнения и маршрутизация биологического материала.
  • Презентация ГВ.

Красные зоны

Риски
  • Контракт с подрядчиком ЛИС-Брегис действует до 15 апреля 2026 года.
  • Процент не сгенерированных ТАП от общего числа — 80%.
  • Не настроен запрет внесения результатов анализов в МИС, выполненных вручную.
  • Не настроен запрет назначения лабораторных услуг на закрытые лаборатории.
  • Отсутствует интеграция лабораторного оборудования, предоставленного по ДБП, в УРО. Требуется доработка, срок не установлен.
  • Не завершена интеграция результатов лабораторных исследований из базы внешнего подрядчика.
  • Не завершена интеграция модуля ЛИС «Склад» с ЕИС закупки; сдвиг сроков: март → сентябрь 2026 года. Ответственный — подрядчик по ГК.
  • Отсутствуют НПА в рамках плана централизации на 2026 год по направлениям СПИД и КВД; контрольный срок — 13.03.2026.

Проблематика

Требует внимания
  • 35 задач просрочено в Битрикс, из них 2 задачи находятся у главных врачей.
  • Протокольные поручения не отрабатываются ответственными лицами МО в установленный срок.
  • Ответственные лица ПВБ/ППБ предоставили данные по 74 из 75 МО.
  • Ответственные по внесению пациентов в ЕМИАС предоставили данные по 73 из 75 МО.
  • Ввиду неготовности Королёвской Б централизация МОЦОМД и Котельниковской П перенесена на II квартал 2026 года.
  • Не выполнен расчет услуг ТПГГ по 4 ЦКДЛ из 8.
